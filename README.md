# SmartGroceries

A mobile-first web app for managing your grocery shopping list, built with React + TypeScript.

## Features

- 📋 **Product list** — add and manage a master list of products
- 🛍 **Shopping list** — select products to buy, with a quick toggle
- ✅ **Mark as bought** — tap items in the shopping list to mark them as purchased
- 💾 **Persistent storage** — data is saved in `localStorage`

The app is automatically deployed to GitHub Pages on every push to `main` via GitHub Actions.
